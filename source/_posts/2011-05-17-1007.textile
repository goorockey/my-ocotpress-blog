1、先下载并安装Live Writer，地址如下
<blockquote><strong>For Xp</strong> ：   <a href="http://explore.live.com/windows-live-writer-xp">http://explore.live.com/windows-live-writer-xp</a></blockquote>
<blockquote><strong>For Other</strong>：<a href="http://explore.live.com/windows-live-writer?os=other">http://explore.live.com/windows-live-writer?os=other</a></blockquote>
2、添加用户：

(1) 选择“其他日志服务”

<a href="http://www.goorockey.com/uploads/2011/05/image11.png"><img class="alignnone size-medium wp-image-77045" title="image1" src="http://www.goorockey.com/uploads/2011/05/image11-300x254.png" alt="" width="300" height="254" /></a>

<!--more-->

(2)设置博客地址、用户名和密码
<blockquote>用户名和密码是Micolog管理页面中设置的rpc用户和密码</blockquote>
<a href="http://www.goorockey.com/uploads/2011/05/image21.png"><img class="alignnone size-medium wp-image-77046" title="image2" src="http://www.goorockey.com/uploads/2011/05/image21-300x256.png" alt="" width="300" height="256" /></a>

(3) 设置博客的类型
<blockquote>推介选择WordPress 2.2+

发布的URL就是 “你的博客地址/rpc”</blockquote>
<a href="http://www.goorockey.com/uploads/2011/05/image31.png"><img class="alignnone size-medium wp-image-77047" title="image3" src="http://www.goorockey.com/uploads/2011/05/image31-300x255.png" alt="" width="300" height="255" /></a>

(4)之后，等它配置完成就可以用Live Writer写Micolog了：）

3、要注意的地方：

(1)更改了博客主题后，要及时更新Live Writer的账户配置
<blockquote>【日志】-【编辑日志设置】，选择“更新账户配置”。</blockquote>
(2)设置自动保存和标题提醒
<blockquote>【工具】-【选项】-【编辑】，勾选“自动保存草稿间隔”</blockquote>
<a href="http://www.goorockey.com/uploads/2011/05/image41.png"><img class="alignnone size-medium wp-image-77048" title="image4" src="http://www.goorockey.com/uploads/2011/05/image41-300x285.png" alt="" width="300" height="285" /></a>
<blockquote>再选择【首选项】标签，勾选“发布前提醒我指定标签”</blockquote>
<a href="http://www.goorockey.com/uploads/2011/05/image51.png"><img class="alignnone size-medium wp-image-77049" title="image5" src="http://www.goorockey.com/uploads/2011/05/image51-300x287.png" alt="" width="300" height="287" /></a>

(3)添加Ping服务器

为了让新博文发布后能被人即时知道，可以设置Ping服务器
<blockquote>【工具】-【选项】-【Ping服务器】</blockquote>
以下是我的Ping服务器：
<blockquote><a href="http://www.feedsky.com/api/RPC2">http://www.feedsky.com/api/RPC2</a>
<a href="http://ping.baidu.com/ping/RPC2">http://ping.baidu.com/ping/RPC2</a>
<a href="http://blogsearch.google.com/ping/RPC2">http://blogsearch.google.com/ping/RPC2</a>
<a href="http://www.zhuaxia.com/rpc/server.php">http://www.zhuaxia.com/rpc/server.php</a>
<a href="http://www.xianguo.com/xmlrpc/ping.php">http://www.xianguo.com/xmlrpc/ping.php</a>
<a href="http://so.blog.qihoo.com/pingblog.html">http://so.blog.qihoo.com/pingblog.html</a>
<a href="http://api.my.yahoo.com/RPC2">http://api.my.yahoo.com/RPC2</a>
<a href="http://search.msn.com/docs/submit.aspx">http://search.msn.com/docs/submit.aspx</a>
<a href="http://www.sogou.com/feedback/blogfeedback.php">http://www.sogou.com/feedback/blogfeedback.php</a></blockquote>
Ping服务器的添加是在精不在多，太多无谓的Ping服务器只会徒增博文发布的时间。
