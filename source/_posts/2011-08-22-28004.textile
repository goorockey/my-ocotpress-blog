2、用移位实现

与很多优化算法相似，用2次幂实现加速。

考虑到x是y的倍数，设x = y * k

因为我们可以用二进制表示任意整数，所以任意整数都可表示成2次幂的和，即：
<blockquote>k = 2^t1 + 2^t2 + …. + 2^tn;</blockquote>
所以有x = y * (2^t1 + 2^t2 + … + 2^tn)，即我们要的结果就是2^t1 + 2^t2 + … + 2^tn

由此，我们可以先找到一个刚好不大于x的s1 = y*(2^t1)，即有y*2^t1 &lt;= x &lt; y*2^(t1+1)，

然后令x2 = x - s1 = y * (2^t2 + … + 2^tn)，从而继续递归直到xn – sn = 0。
